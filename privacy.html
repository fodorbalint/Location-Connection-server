If you choose to use my service, then you agree to the collection and use of information in relation to this policy. The personal information that I collect is used for providing the service. I will not use or share your information with anyone except as described in this Privacy Policy.

Information collection and use

For both registered and non-registered users, the app will ask your consent to use location if you wish to take advantage of this feature.
Non-registered users will send the last acquired position to my server when requesting a list of registered users. This will be used for calculating distance between you and them, so it can be displayed on their profile page, or used for distance filtering.
It is only stored on my server in the HTTP logs which are periodically deleted. Location is only acquired while you are using the app.
You have the option to set an address if you do not wish to use your own location.

When registering a new account, the app sends the given data to my server, including the pictures you upload, and they will be stored, so other users can find you.
At any time you have the option to turn the location on or off, and set sharing levels such as:
- sharing location with everyone
- sharing location with users you liked
- sharing location with users you matched with
- sharing location with users you added as friend
- not sharing it with anyone
The same applies to distance.

Location is acquired by default both while you are using the app and in the background. Background access can be turned off in the app's Settings.

A list of locations acquired in the last 24 hours is stored on your device, and displays as a timeline in the app's Location History page.
The server however only keeps the last location data of registered users. Turning off location in your account deletes these data from the server too.

Your matches and conversation history is stored on the server indefinitely, and if you unmatch from someone, the match will disappear from your account. It is deleted from the server when both parties choose to do so.
When deleting your account, all records of it are removed from the server, except the conversation history of your matches if the other person has not unmatched yet.

Service Providers

The app uses Firebase Cloud Messaging to transmit messages and notifications. It works by accessing your device identifier, which is then sent to my server for use when sending data to your device.
They are obligated to perform the tasks assigned to them, and not to disclose or use the information for any other purpose.
